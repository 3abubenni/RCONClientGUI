### RCON Client GUI

A lightweight desktop application for connecting to game servers via the RCON (Remote Console) protocol. 
Execute commands and manage your server through a simple graphical interface.

### Features

- **Command History** - Automatically saves command history per server connection
- **Validation** - Prevents empty hostnames, ports, and passwords
